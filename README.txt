STEP1# Install and open Rufus "https://rufus.akeo.ie/"
STEP2# Select flash drive to wipe and install
STEP3# Select MBR partition scheme for BIOS or UEFI
STEP4# Select NTFS as File System
STEP5# Have the following selected ONLY "Quick Format" "Create a bootable disk image using _(ISO IMAGE)_" "Create extended label and icon files"
STEP6# Next to _(ISO IMAGE)_ press the button and select the file "Ghose.iso"
STEP7# Press Start!
STEP8# DragnDrop/Copy paste the "ControlComputer.GHO" to the root of the newly made flash drive.

THATS IT! 
Now boot to the flash drive from the BIOS and install the image. :)
